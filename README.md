🏛️ AI-Based Virtual Reconstruction of Architectural Ruins

📌 Problem Statement

To develop an AI-based system capable of reconstructing partially destroyed or ancient architectural ruins by generating visually accurate and stylistically consistent restorations.

💡 Solution

We designed a pipeline integrating multiple machine learning models to reconstruct architectural structures from ruin images. The system ensures that the generated output closely matches the original architectural style.

⚙️ Models Used

* Stable Diffusion v1.5
* ControlNet
* Canny Edge Detection (for structural guidance)

🚀 Approach

1. Input image of architectural ruins
2. Apply Canny Edge Detection to extract structural outlines
3. Use ControlNet to guide structure-aware generation
4. Generate reconstructed output using Stable Diffusion v1.5
5. Produce visually enhanced and realistic architectural reconstruction

📊 Results

* Generated high-quality reconstructed images
* Preserved architectural style and structural integrity
* Demonstrated strong potential for heritage restoration applications

🌟 Key Highlights

* Multi-model pipeline approach
* Structure-preserving reconstruction using edge detection
* Realistic output generation using diffusion models
